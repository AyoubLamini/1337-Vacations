Frontend project for vacation management built during my internship as part of my training at 1337 (42 Network) in Khouribga, Morocco — a peer-learning programming school where students work on real projects and gain professional experience.

This application serves as a vacation management interface for 1337 students, allowing them to browse and manage vacation requests. It was my first project using React.js, and it helped me learn modern frontend development concepts including hooks, state management, theming, and responsive UI design.

🚀 Features

✅ Interactive user interface built with React.js
✅ Supports both Light Mode and Dark Mode
✅ Responsive layout with pure HTML & CSS
✅ Built using React Hooks (such as useState, useEffect)
✅ Practical experience with real-world frontend application workflows

🛠 Technologies Used

React.js – UI library for building component-based user interfaces

JavaScript (ES6+) – Core language for logic & interactivity

HTML & CSS – Structure and styling of the app

React Hooks – For managing component state & lifecycle

📦 Installation

To run this project locally:

# Clone the repository
git clone https://github.com/mza7a/1337-vacations.git

# Navigate into the project directory
cd 1337-vacations

# Install dependencies
npm install

# Start the development server
npm start

After running npm start, the app will open in your default browser (usually at http://localhost:3000).

=======
1337-Vacations is a frontend application developed during my internship at 1337 (42 Network) in Khouribga, Morocco. The project provides a vacation management interface that allows students to browse and manage vacation requests through a modern and responsive web application.

As my first project built with React, it served as an introduction to component-based development, state management, React Hooks, and user interface design.

Overview

The application focuses on providing a simple and intuitive experience for managing vacation-related information while maintaining a clean and responsive interface across different devices.

Features
Responsive user interface built with React
Light and dark theme support
Component-based architecture
State management using React Hooks
Mobile-friendly design
Clean and modern user experience
Technologies
React.js
JavaScript (ES6+)
HTML5
CSS3
